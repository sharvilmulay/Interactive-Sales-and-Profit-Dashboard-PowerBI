# Interactive-Sales-and-Profit-Dashboard-PowerBI-
● Project Overview

This project is a Data Analytics Mini-Project created using Power BI. It transforms raw sales data into an interactive visual report to help businesses understand their sales trends, product performance, and profitability.

● Key Features

- Executive Summary: Large cards showing Total Sales and Profit at a glance.

- Sales Trends: A line chart showing how sales change month-by-month.

- Product Analysis: Bar charts identifying top-performing products and categories.

- Profitability Insights: A scatter plot showing the relationship between revenue and profit.

- Interactive Filters: Slicers to filter the entire dashboard by Region and Date.

● Tech Stack

- Tool: Power BI Desktop

- Data Source: Excel/CSV ( train.csv )

- Techniques: Data Cleaning (Power Query), Data Modeling, and DAX (Calculated Measures).

● How to View

1. Download the ' PowerBI Sales Dashboard Project.pbix ' file.

2. Open it using Power BI Desktop.

3. Alternatively, view the 'Report View SS.png' for a static preview OR Open 'PBI Dashboard.pdf' for more clear image.
